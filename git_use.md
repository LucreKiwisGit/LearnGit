# Git 使用手册

![](E:\Learn\Git's Structure.png)

```
Remote:远程仓库
Repository:本地仓库
Index:缓冲区
Workspace:工作区
```

### 一、基本设置

```bash
配置个人信息
git config --global user.name "Lucre"
git config --global user.email "LucreKiwis@gmail.com"

创建本地仓库
git init
```
